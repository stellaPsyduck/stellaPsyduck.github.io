Task Allocator

Summer 2025

Project Goal

    The goal of this project is to develop a program that helps a group of people allocate tasks based on individuals’ willingness to perform specific tasks and the importance of the task. I aim to apply this to two situations. The first is a house cleaning scenario, where roommates can divide up their weekly chores based on preference, ensuring that every task gets completed at least every two weeks. The second scenario is inspired by task organization on the robotics team. It sets up a more complex scenario because it involves more conditionals. This task allocator must divide tasks based on preference, skill, number of people required for each task, and time people have available. The first scenario should solely be able to use maximum flow algorithms. However, the extra constraints in the second scenario make it more easily solved with a linear programming approach.
